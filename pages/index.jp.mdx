# OpenMLガイド

オープンソース機械学習の世界へようこそ!私たちの使命は、個人が人工知能と機械学習の領域へのエキサイティングな旅に乗り出せるようにすることです。この Web サイトは、この分野で最も評判の高い機関や組織から提供される知識、洞察、リソースの宝庫への入り口です [Stanford Online](https://online.stanford.edu/), [DeepLearning.AI](https://www.deeplearning.ai/), [Fast.AI](https://www.fast.ai/), [3Blue1Brown](https://www.youtube.com/@3blue1brown) 他にもたくさんあります。

![Geometric Neural Network](/static/img/neuralNetwork.png)

進化し続けるテクノロジー環境において、機械学習は変革の力として機能し、新たな可能性を解き放ち、世界の産業を再構築します。経験豊富なデータ サイエンティストであっても、熱心に学習している人であっても、ML の世界にまったく慣れていない人であっても、当社のプラットフォームはあらゆる専門知識レベルに対応します。私たちの目標は、機械学習の広大な世界へのアクセスを民主化し、誰もがアクセスできるようにすることです。

ここでは、無料のチュートリアル、ガイド、記事、コース、プロジェクトの厳選されたコレクションが見つかります。これらはすべて、複雑な概念をわかりやすくし、ML の世界で優れているために必要なスキルを身につけるように設計されています。

コンピュータ ビジョン、自然言語処理、強化学習、ディープ ラーニング、またはその他の機械学習の側面に興味がある場合でも、ニーズに合わせた豊富なリソースが見つかります。私たちの使命は、情報を提供するだけではなく、このエキサイティングな旅で互いにサポートし、刺激し合う学習者のコミュニティを育成することです。

### このウェブサイトの操作方法

- カタログを参照: ML リソースの広範なライブラリを探索してください。コース、講義、チュートリアルなどが見つかります。

- あなたの道を選択してください: 初心者でも経験豊富な実践者でも、私たちはあなたのニーズに合わせたリソースを用意しています。あなたの目標と専門知識レベルに合わせたコースを選択してください。

- オープンソース、オープンアクセス: 当社のコンテンツはすべてオープンソースであり、自由にアクセスできます。ペイウォールはなく、探索されるのを待っている知識だけがあります。

- コミュニティに参加: [Discord](https://discord.gg/vp4E46d3jB) サーバーで他の学習者やメンターとつながりましょう。活気に満ちたコミュニティを通じて、コンセプトについて話し合ったり、質問したり、ML プロジェクトで協力したりできます。
